## Tech Stack
- Vue3.js
- Tailwindcss
- AOS


## Run Locally

Clone the project

```bash
  git clone git@github.com:bartoginski/cv.git
```

Go to the project directory

```bash
  cd cv
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run serve
```

After a while go to [http://localhost:8000](http://localhost:8000).
